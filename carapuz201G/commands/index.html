Commands

Main slash and prefix command overview

Moderation

/warn member reason

Issue a warning to a member and increase their server warn count.

/mute member time reason

Timeout a user for a specific duration.

/kick member reason

Kick a member from the guild.

/ban member reason

Ban a member from the guild.

/clear amount

Bulk delete recent messages from the current channel.

/clearuser member amount

Delete recent messages from a specific member.

/clearwarns user amount

Remove warns from a member.

!warn / !mute / !kick / !ban

Prefix moderation duplicates for reply-based moderation workflows.

Kingdoms

/kingdom_create kingdom_name

Create a kingdom for the current guild.

/kingdom_info

Show current kingdom statistics.

/kingdom_upgrade stat

Upgrade defense, walls or towers.

/kingdom_train unit_type amount

Train warriors, archers or mages.

/kingdom_raid target_guild_id

Attack another guild kingdom.

/bank

Manage the guild treasury.

/invest amount target_guild

Invest in another kingdom.

/withdraw_invest target_guild

Withdraw your investment.

/countdown seconds

Run a spoken countdown.

/time city_or_timezone

Speak current time in a selected timezone or city.

/weather city

Get and optionally speak weather information.

/carapuz_radiofm

Start radio mode if supported by the host.
