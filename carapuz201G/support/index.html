Official Support Server

The fastest way to get support is through the official Discord support server.

Join the support server

In-Bot Support

You can use the /support command inside Discord to send a support request directly to the bot owner.

Bug Reports

If you report a bug, it helps to include:

  • the command used,
  • what you expected to happen,
  • what actually happened,
  • error text or screenshot if available,
  • whether the issue happened on a single server or multiple servers.

Privacy Requests

If you want to remove your data from the bot, use the /deletedata command where available. You may also contact support if you need clarification regarding stored bot data.
