Support

How to get help and contact the project

Official Support Server

The fastest way to get support is through the official Discord support server.

discord.gg/pBh4rRK3TT

In-Bot Support

You can use the /support command inside Discord to send a support request directly to the bot owner.

Bug Reports

If you report a bug, it helps to include the command used, expected result, actual result, and any error text or screenshot.

  • What command you used
  • What you expected to happen
  • What actually happened
  • Any visible error message or screenshot
  • Whether the issue affects one or multiple servers

Privacy and Data Requests

If you want to remove your data from the bot, use the /deletedata command where available. You may also contact support for clarification about stored bot data.

For privacy-related questions, always use the official support paths listed on this page.
