Features

What the bot can do

Moderation

Warnings, mutes, bans, anti-raid tools, filters and logging systems.

💰

Economy

Daily rewards, jobs, casino, shop, ratings, quests and investments.

🏰

Kingdoms

Create kingdoms, train armies, raid others, manage treasury and investments.

🎫

Tickets

Support panels, topic selection, claiming, closing, reopening and transcript export.

✅

Verification

Captcha verification, Roblox verification and Steam verification with roles.

🔔

Utility

Reminders, translation, search, dictionary, calculators, polls and embeds.

🔊

Voice Features

Music, speech, countdowns, radio and private voice room management.

🧩

Server Tools

Custom commands, reaction roles, welcome cards, stats channels and cloning tools.

🧠

Local AI (Optional)

Optional Ollama-powered local AI features intended only for bot functionality.

Feature Philosophy

Carapuz201G is built to combine administration and community tools in one place. Instead of splitting moderation, utility, economy and voice into separate bots, the project aims to keep the server experience centralized.
