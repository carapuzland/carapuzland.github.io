Features

What the bot can do

🛡️

Moderation

Warns, mutes, kicks, bans, clear commands, message logs, appeals, text filters and anti-spam.

💰

Economy

Daily rewards, jobs, transfers, shop, casino, inventory, ratings and quests.

🏰

Kingdoms

Create kingdoms, train armies, raid other kingdoms, manage treasury and investments.

🎫

Tickets

Support ticket panels, topic selection, claiming, closing, reopening and transcript export.

✅

Verification

Captcha verification, Roblox verification and Steam verification with roles.

🔔

Reminders & Utility

Reminders, embeds, translation, search, dictionary, password generation and calculators.

🔊

Voice Features

Music playback, voice speech, countdowns, radio mode and private room management.

🧩

Server Tools

Custom commands, reaction roles, welcome cards, stats channels, polls and cloning tools.

🧠

Local AI (Optional)

Optional Ollama-powered local AI functionality designed for feature execution only, not user-data training.

Feature Philosophy

Carapuz201G is built to combine both administration and community features in one place. Instead of splitting utility, moderation, economy and voice into separate bots, the project aims to keep the server experience centralized.
