Privacy Policy

How Carapuz201G handles data

1. Introduction

This Privacy Policy explains how Carapuz201G collects, stores, uses, and deletes data when users interact with the Bot through Discord.

By using the Bot, you acknowledge this Privacy Policy. If you do not agree, do not use the Bot.

2. Data We Collect

The Bot may collect only the data reasonably necessary for its functionality, including:

  • Discord user IDs for moderation, economy, reminders, verification and bot systems.
  • Discord guild IDs and configuration data for server-specific bot settings.
  • Economy-related values such as balances, workers, utility levels, quests and investments.
  • Moderation data such as warning counts and related guild settings.
  • Temporary verification session data for Roblox / Steam verification flows.
  • Temporary encrypted anti-raid backups containing limited server metadata and recent message snapshots.
  • Opt-in cmail preference data.
The Bot is designed not to request or collect passwords, Discord tokens, or login credentials.

3. How We Use Data

  • To provide moderation, economy, verification, profile, reminder, giveaway, utility and support features.
  • To store per-server settings chosen by guild administrators.
  • To support anti-raid restore features using temporary encrypted backups.
  • To maintain, secure and improve the intended functionality of the Bot.
We do not sell, rent, broker, or monetize Discord API data.

4. Ollama / Local AI

Some builds of the Bot may optionally use a locally-hosted Ollama model for feature-specific processing.

  • When used, local AI is intended only to provide requested bot functionality.
  • Discord user message content is not used to train, fine-tune, or improve the model.
  • The Bot is not intended to create AI training datasets from Discord user content.
Local AI processing is intended for feature execution only, not for model training on Discord user data.

5. Data Sharing

We do not share data with ad networks, data brokers or monetization services.

Data may only be disclosed in limited cases such as legal requirements, requested third-party functionality, or explicit user-requested actions.

6. Storage and Security

Persistent bot data is stored with encryption at rest.

Depending on deployment, data may be stored on the operator’s computer, on a private server, or on a hosting environment used to run the Bot.

7. Temporary Backups

Anti-raid protection may create encrypted temporary backups containing limited server metadata and recent message snapshots strictly for restoration and recovery purposes.

8. User Rights and Deletion

Users may request deletion of their bot-related data through tools such as /deletedata.

9. Cmail / User Messaging

The Bot may provide an optional user-to-user messaging feature (cmail).

  • Users must explicitly opt in before receiving cmail.
  • Users can disable cmail at any time.
  • The feature is not intended for unsolicited advertising or spam.

10. Children’s Privacy

The Bot is not directed to children under 13. Use of the Bot must comply with Discord’s minimum age requirements and applicable law.

11. Changes to This Policy

We may update this Privacy Policy from time to time. Continued use of the Bot after changes means you accept the updated version.
